# Digital-Twin-Driven-Life-Cycle-Energy-Optimisation-and-Dynamic-Scheduling-for-Heavy-Equipment
MATLAB code and Simulink model for simulating a large hydraulic forging press.
The "run_experiments_v1.m" works with the Simulink model "LHP_DT_v1.slx" to evaluate single-machine effects of load, speed, temperature and operating mode.
Use "main.m" to simulate system-level scheduling, degradation and disturbance response.

├── results/               
│   ├── Tables/     # Tables generated by "run_experiments_v1.m"
│   └── Figs        # Figures generated by "run_experiments_v1.m"
├── models/         # Simulink model
